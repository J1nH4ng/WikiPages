Docker 容器仓库搭建 ​

对于企业而言，Docker 官方的 Docker Hub 有两个弊端：

  • 服务器建于国外，访问速度受限
  • 数据安全性有待考量

所以，在很多时候不适合去使用 Docker Hub，而是选择自己搭建私有化的 Docker 容器仓库。

这里我们使用 CNCF 基金会开源的 Harbor 作为私有化的 Docker 容器仓库，Harbor 在物理机上的安装部署是比较困难和复杂的，而为了简化安装，Harbor 官方直接将 Harbor 制作成了在 Docker 中运行中的容器。

所以整个 Harbor 服务器需要的软件如下：

安装配置 Docker ​

安装 ​

YUM 安装 ​

  1. 先添加 Docker 官方仓库

    bash
    yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
    1

  2. 修改源文件

    由于并不是原生的 CentOS 系统，所以需要手动指定源文件中的版本以下载正确的 Docker

    bash
    vim /etc/yum.repos.d/docker-ce.repo
    1

    执行如下命令批量查询替换

    bash
    :1,$s/\$releasever/7/g
    1

  3. 查询 Docker 版本

    bash
    yum list docker-ce --showduplicates | sort -r
    1

  4. 安装 Docker

    bash
    yum -y install docker-ce
    1

  5. 启用 Docker

    bash
    systemctl enable docker --now
    1

二进制安装 ​

  1. 下载 Docker 二进制文件

    bash
    cd /usr/local/src/

wget -c https://download.docker.com/linux/static/stable/x86_64/docker-28.1.1.tgz
    1

  2. 解压该文件并授予权限

    bash
    tar -zxvf docker-28.1.1.tgz

chown root:root docker/*
    1
    2
    3

  3. 移动至环境变量中

    bash
    cp -p docker/* /usr/bin/
    1

    小技巧：

    cp -p 命令的 -p 选项表示保持源文件属性，包括权限、时间戳等。

  4. 创建 Docker 用户组

    bash
    groupadd docker

SocketUser=root

[Install]
WantedBy=sockets.target

配置 ​

Docker 安装完成之后，还需要配置 daemon.json 文件，用于实现如下修改：

  • 存储位置修改
  • Docker 容器内 IP 变更
  • 日志大小变更
  • 私有仓库配置
  • 镜像仓库配置

修改 daemon.json 文件，如果没有则创建：

bash
mkdir -p /etc/docker

vim /etc/docker/daemon.json
1
2
3

写入如下内容：

json
{
  "data-root": "/data/docker",
  "registry-mirrors": [
    "https://dp.4r3al.team"
  ],
  "bip": "172.18.1.1/24",
  "insecure-registries": ["192.168.50.120:3380"],
  "log-opts": {
    "max-size": "150m",
    "max-file":"3"
  },
  "exec-opts": ["native.cgroupdriver=systemd"]
}

安装 Docker Compose ​

Harbor 安装调用的是旧版的 docker-compose 命令，所以需要安装最新版的 docker-compose，执行如下命令进行安装：

bash
curl -SL https://github.com/docker/compose/releases/download/v2.35.0/docker-compose-linux-x86_64 -o /usr/local/bin/docker-compose

chmod +x /usr/local/bin/docker-compose

ln -s /usr/local/bin/docker-compose /usr/bin/docker-compose

安装配置 Harbor ​

安装[1] ​

  1. 下载 Harbor 离线安装包：

    bash
    wget https://github.com/goharbor/harbor/releases/download/v2.13.0/harbor-offline-installer-v2.13.0.tgz
    1

  2. 解压安装包：

    bash
    tar -zxvf harbor-offline-installer-v2.13.0.tgz
    1

配置[2] ​

  1. 复制配置文件模板

    bash
    cd harbor/

cp harbor.yml.temp harbor.yml
    1
    2
    3

  2. 修改如下配置内容：

    diff
    hostname: ${HARBOR_SERVER_IP}

# 注释掉如下内容
# https:
#   port: 443
#   certificate: /path/to/certificate
#   private_key: /path/to/private_key

harbor_admin_password: admin123
harbor_admin_password: ${HARBOR_ADMIN_PASSWORD}

database:
  password: root
  password: ${HARBOR_DATABASE_PASSWORD}

data_volume: /data
data_volume: ${HARBOR_DATA_VOLUME}
